# Contributing to QIE

Thank you for your interest in QIE! 

At this time, **QIE is not accepting external contributions**. The project is in its early stages and the codebase is still stabilizing. Once the plugin is hosted, functional, and has grown a bit, contributions will be welcomed.

In the meantime, you're welcome to:

- ⭐ **Star** the repo to show interest
- 🐛 **Open issues** if you find bugs or have feature ideas
- 👀 **Watch** for updates as development progresses

Check back later — this policy will be updated when the project is ready for contributors.
